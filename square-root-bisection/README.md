# Square Root Finder using Bisection Method

This project calculates the square root of a number using the **Bisection Method**, a numerical algorithm based on binary search.

## Features
- User input support
- Modular Python structure
- Handles invalid inputs
- Uses tolerance-based approximation
- Detects convergence
