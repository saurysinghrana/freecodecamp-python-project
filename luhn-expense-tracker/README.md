# Luhn Algorithm Card Validator

A Python program that validates card numbers using the **Luhn Algorithm**.

## Features
- Validates card numbers
- Supports numbers with spaces or hyphens
- Interactive command-line interface
- Implements checksum verification

## Concepts Used
- String Slicing
- Loops
- Functions
- Conditional Statements
- Digit Manipulation
- Algorithms

## Example Run

```text
=== Luhn Algorithm Card Validator ===

Enter card number: 4111-1111-4555-1142

VALID card number.
```
